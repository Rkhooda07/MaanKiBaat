### 2. Get OpenAI API Key (Easiest Option)
- Go to https://platform.openai.com/api-keys
- Sign up/Login to OpenAI account
- Create new API key
- Copy the API key (starts with sk-...)

### 3. Environment Setup
- Create .env file in project root
- Add: `OPENAI_API_KEY=your_openai_key_here`

### Alternative: Google AI API
- If you prefer Google AI: https://aistudio.google.com/
- Use: `PALM_API_KEY=your_google_key_here`
